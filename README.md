# LandServer
该存储库存储Land服务器的一些内容，包含资源包，数据包，插件等。
2024年3月22日，Land服务器上架MC百科，真正地公开了服务器。

（本页面介绍Land服务器的主服：Minecraft原版生存&创造服）

## 概述
Land是一个Minecraft Java版服务器，目前版本为1.21.11，使用了Leaf服务端。服务器启用在线模式，玩家可以使用**正版，基岩版，LittleSkin账户**加入服务器。

截至2026年2月12日，Land共有23.32GB的数据，约389名玩家加入了服务器，运行51个插件，群聊含有76个人。

## 内容
Land虽然是个小服，但内容却不少。

服主写了一个数据包，可以**腐肉变皮革**，石头快速烧制，合成附魔金苹果，猪灵的头烧成下界合金剑；还有两种**资源箱**，可以在主城村民商店换到；使用1.21.6以上版本加入服务器，打开游戏菜单还有**快捷指令**选项！

服主还制作了资源包（现已停更），玩家可以投稿歌曲在服务器播放。

2025年暑假，Land新增**CreateLand世界**，本世界数据和生存世界隔离。玩家终于可以在Land玩到创造模式了！除此之外，服务器还有**放大化世界**。这些扩展世界由插件[https://modrinth.com/plugin/multiverse-core Multiverse-Core]实现。
